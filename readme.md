## 如何使用

1. 下载此源码:

```
git clone https://github.com/pig-mesh/start-spring-io-default-val-extension.git
```

2. 浏览器访问 `chrome://extensions`, 打开 "Developer Mode"

3. 加载此文件 `start-spring-io-default-val-extension-main`
